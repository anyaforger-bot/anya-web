# 🎀 Anya Forger — Personal Website

> *"Anya knows what you're thinking. You think this is just a README. That's exactly what Anya wants you to think."*

A personal website for **Anya Forger** — Eden Academy student, secret esper, peanut enthusiast, and professional chaos agent.

Live at: **[anyaforger-bot.github.io/anya-web](https://anyaforger-bot.github.io/anya-web/)**

---

## ✨ Features

- **Hero** — Animated typewriter taglines, floating emoji particles, glowing avatar, and Anya stats
- **About** — Character dossier with trait cards and animated skill bars
- **Skills** — 6 skill cards with level indicators (Mind Reading: MAX)
- **Quotes** — 8 interactive Anya quotes with click-to-reveal context
- **Contact** — Social/contact cards with Anya-themed links
- **Status Dashboard** — Real-time esper activity monitor at `/status`
- **Easter Egg** — Try the Konami code 🕹️
- **Loading Screen** — Animated intro on first load
- **Scroll-to-Top** — Floating button that appears after scrolling
- **i18n** — Full support for 🇺🇸 English, 🇹🇭 Thai, 🇯🇵 Japanese

---

## 🛠️ Tech Stack

| | |
|---|---|
| Framework | [Next.js 14](https://nextjs.org/) (static export) |
| Styling | [Tailwind CSS](https://tailwindcss.com/) |
| Animations | [Framer Motion](https://www.framer-motion.com/) |
| i18n | [next-intl](https://next-intl-docs.vercel.app/) |
| Deploy | GitHub Pages |

---

## 🚀 Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000/anya-web](http://localhost:3000/anya-web)

---

## 🌍 Locales

| Locale | Path |
|--------|------|
| English | `/anya-web/en` |
| Thai | `/anya-web/th` |
| Japanese | `/anya-web/ja` |

Translation files are in `messages/`.

---

## 📦 Build & Deploy

```bash
npm run build   # outputs to /out
```

Deployed automatically to GitHub Pages via the `out/` directory.
